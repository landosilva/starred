Favorites tray + selection history for the Unity Editor.

Star the assets and scene objects you live in. Jump back to anything you just touched. See at a glance what's starred — right inside the Project window and Hierarchy.

Install via UPM View on GitHub

Favorites

A tray for the things you actually use.

Tools → Starred → Favorites

Drop in project assets or scene GameObjects. Click to select, double-click to open or frame, drag out onto Inspector fields.

  • Flat ordered list — no folders, no nesting. It stays a tray.
  • Reorder by drag. Lens icon to ping in Project or Hierarchy.
  • Scene-bound entries keyed by scene + hierarchy path — only visible when their scene is loaded.
  • Missing or renamed objects show red so you can clean them up.

Selection History

Back to what you just had.

Tools → Starred → History

Auto-populated, most-recent-first. Re-selecting bumps to the top. Each row has a ★ button to promote straight to Favorites.

  • Tracks both project assets and scene / prefab-stage GameObjects.
  • Single-click inspects. Lens pings. Double-click opens or frames.
  • Size caps configurable: 4 / 8 / 16 / 32.

Star Overlays

See what's starred without opening anything.

Favorited items get a ★ overlay in the Project and Hierarchy windows. Click the overlay to unfavorite directly. Both toggle independently in Preferences.

Context Menu

Everything one right-click away.

  • 👁 Show in Project / Hierarchy
  • 🗂 Show in Explorer
  • ↗ Open — scene, IDE, prefab stage…
  • 📷 Frame in Scene View
  • 📋 Copy Path / GUID / Hierarchy Path
  • ❌ Remove from Favorites

Preferences

Tune it to your workflow.

macOS: Unity → Settings → Starred  ·  Windows: Edit → Preferences → Starred

  • Show star in Project window — toggle Project ★ overlay.
  • Show star in Hierarchy — toggle Hierarchy ★ overlay.
  • Selection history max entries — 4 / 8 / 16 / 32.
