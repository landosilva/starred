# Starred

A compact favorites tray plus selection history for the Unity Editor.

## What you get

- **Favorites** (`Tools → Starred → Favorites`) — drag project assets or scene / prefab-stage GameObjects into a flat tray. Lens pings, double-click opens, drag-out drops into the Inspector. Contextual scene-bound entries only appear while their scene or prefab is open.
- **Selection History** (`Tools → Starred → History`) — auto-populated list of the last things you selected (assets *and* scene objects). Click the star on any row to promote it into Favorites.
- **Star overlay** — small gold ★ on favorited items in both the **Project** window and the **Hierarchy**. Click the star to remove.
- **Preferences pane** — `Edit → Preferences → Favorites & History` toggles the stars and tunes the history size.

## Install

Unity Package Manager → **Add package from git URL…** and paste this repo's URL.

Or clone locally and add via **Add package from disk…** pointing to the `package.json`.

## Requirements

- Unity **6.0** or newer.
